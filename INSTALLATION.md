# ABAP Development Project - Expense ALV Report

## Quick Start

This repository contains a professional SAP ABAP project for expense management.

### Requirements
- SAP System (ECC/S4HANA)
- ABAP Workbench access
- SAP GUI or WebGUI

### Files Included
- `src/z_expense_alv.abap` - Main ABAP report source code
- `README.md` - Project overview and features
- `docs/TECHNICAL_DOCUMENTATION.md` - Detailed technical documentation

### How to Use

1. Copy the ABAP code from `src/z_expense_alv.abap`
2. Open your SAP system and navigate to SE80
3. Create a new Report: z_expense_alv
4. Paste the code and activate
5. Execute the report

Full documentation available in README.md and technical docs folder.
